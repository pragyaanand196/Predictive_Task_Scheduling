Predictive Task Scheduling Mechanics

This system uses a rule-based multi-criteria scheduling algorithm to recommend optimal task owners based on three criteria: Active Workload (lowest current hours is preferred), Historical Completion Rates (highest success counts is preferred), and Specific Task Type Speed (relative ratio of actual hours versus estimated hours). This prevents task bottlenecking and improves completion accuracy.

1

Capacity (40%)

Calculates total backlog workload (Pending/In-Progress) to avoid overloading employees.

2

Reliability (30%)

Measures the historical percentage of successfully completed tasks versus assigned tasks.

3

Efficiency (30%)

Tracks the employee's speed ratio (Actual/Estimated) specifically for the selected task type.
